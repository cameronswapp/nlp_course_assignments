# Course Assignments for Intro to NLP
This repository contains assignments related to DSAI 6810 (Intro to NLP) taught at Utah State University in the Fall semester 2025. The [agent_project folder](./agent_project/) contains the final project for the course, which covered most of the topics covered throughout the semester. This folder contains an AI agent designed to help users gain information about Aggie sporting events. It uses tools to read from a database, perform retrieval-augmented generation, and search the web.

Folders beginning with a number indicate specific units and assignments that were covered in the course. Each of these folders contain the assignment corresponding to that unit along with any data that was used for the completion of the assignment.
